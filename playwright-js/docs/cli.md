# Run E2E Test

```sh
npx playwright test
npx playwright test --ui                  # UI Mode
npx playwright test --project=chromium    # Chromium
npx playwright test example               # Test Specific File
npx playwright test --debug               # Debugging Mode
npx playwright codegen                    # Code Generating Mode 
npx playwright test --headed --workers 1  # Set Headed and Workers
```
